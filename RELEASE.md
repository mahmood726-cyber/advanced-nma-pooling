# Release Process

This project uses a tag-driven GitHub release flow for reproducibility and
provenance.

## Local Maintainer Steps

1. Run the full test suite.
2. Build the distribution artifacts.
3. Generate the release metadata files.
4. Commit the release-ready tree.
5. Create an annotated tag.
6. Push the branch and tag to GitHub.

Example commands:

```bash
pytest -q
python -m build --no-isolation
python -m twine check dist/*
python scripts/release_manifest.py --dist-dir dist --output-dir dist --version 0.1.1
git add .
git commit -m "Release 0.1.1"
git tag -a v0.1.1 -m "Release v0.1.1"
git push origin main --follow-tags
```

## GitHub Release Automation

The release workflow lives at `.github/workflows/release.yml`.

When a `v*` tag is pushed, the workflow:

1. Runs `pytest -q` on Python 3.11, 3.12, and 3.13.
2. Builds the source distribution and wheel.
3. Runs `twine check` on the built artifacts.
4. Generates `dist/SHA256SUMS.txt` and `dist/release-manifest.json`.
5. Uploads the release artifacts to the workflow run.
6. Generates a GitHub artifact attestation for the built files.
7. Publishes a GitHub release from the existing annotated tag.

## Provenance Model

- Local release integrity is captured with `SHA256SUMS.txt` and
  `release-manifest.json`.
- Remote build provenance is captured by GitHub artifact attestations generated
  in the release workflow.
- Consumers can verify release provenance with GitHub CLI, for example:

```bash
gh release verify v0.1.1
gh release verify-asset v0.1.1 ./nma_pool-0.1.1-py3-none-any.whl
```

## Signing Note

This workspace does not currently have `gpg` or local `sigstore` tooling
configured, so local detached signatures are not part of the default release
process here. The governed path for signed provenance is the GitHub artifact
attestation generated by the release workflow.

## Repository Settings

For the strongest release governance, enable these settings in GitHub:

- Protected default branch
- Required status checks for CI
- Immutable releases
- Artifact attestations
